Foodiepedia Website

http://foodie-pedia.appspot.com/
View on GitHub

Foodiepedia is a website that focuses on different foods, ingredients, recipes and other items which are relevant to foods. User can search for really good recipes, add a link of a good recipe to an item, add an new item, which can be ingredients, wines or anything that is relevant to foods. Also user can search items by product barcode.

I want to make Foodiepedia function as a "food" social network, and users can communicate with each other about cooking, choosing ingredients and restaurants.
User system is almost finished. Now I am adding 'change password' and 'forget password' function to user system.
Website UI Design is processing.
